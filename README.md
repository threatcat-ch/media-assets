## Logo

Prefixes:

| prefix | desc | example |
| --- | --- | --- |
| ``full_`` | Full logo with symbol and logotype | ![](/logo/PNG/full_white_on_red_small.png) |
| ``head_`` | Just the symbol | ![](/logo/PNG/head_white_on_red_small.png) |

| label | desc | example |
| --- | --- | --- |
